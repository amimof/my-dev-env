## What is it
* A terminal [multiplexer](https://en.wikipedia.org/wiki/Multiplexing)

# Features
* Persistence 
* Multiple windows & panes
* Session sharing
* Record the screen to logfile
* Password protect sessions
* Vi-style visual editor
* Verry customizable
